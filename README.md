## Environment vars
| Name                          | Description                               |
| ----------------------------- | ------------------------------------------|
|DATABASE_URL                   | URL of database used to prisma connection |
|JWT_SECRET_KEY                 | Secret key used for jwt                   |

## Pre-requisites
- Install node version 18.12.1

## Getting started

- Install dependencies
```
npm i
```

- Run the project
```
npm run dev:server
```
